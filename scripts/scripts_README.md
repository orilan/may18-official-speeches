# 분석 스크립트 / Analysis Scripts

본 연구의 분석은 Claude Code(Anthropic)를 활용한 대화형 방식으로 수행되었으며, 단일 실행 스크립트 형태의 코드는 현재 공개되어 있지 않습니다.

분석 절차의 상세한 내용은 다음을 참조하십시오:
- 코딩 기준 및 절차: [`codebook/coding_scheme.md`](../codebook/coding_scheme.md)
- 분석 방법 개요: 논문 본문 2장 2절

분석에 사용된 주요 도구 및 라이브러리:
- Python 3.10
- pandas (데이터 처리)
- scipy.stats (통계 분석: chi-square, Fisher's exact test)
- numpy (조정 표준화 잔차 계산)
- KoNLPy (한국어 형태소 분석, 시제 분류)

---

*Analysis scripts are not currently available as the analysis was conducted interactively using Claude Code (Anthropic). For details on the analytical procedure, see [`codebook/coding_scheme.md`](../codebook/coding_scheme.md) and the Methods section (Chapter 2) of the paper.*
